# Shape Reconstruction of Object-Level Building from Single Image Based on Implicit Representation Network

IEEE Geoscience and Remote Sensing Letters

**摘要：**物体级建筑物（SROLB）的三维形状重建是遥感领域的基本问题之一。特别是，利用单个遥感图像（SRSI）进行形状重建，可以在简化输入数据方面提供更好的可扩展性和可转移性。近年来，基于神经网络的形状重建方法得到了广泛的研究。但是，大多数模型生成的曲面不规则，细节很少。此外，SRSI中复杂的背景会导致网络泛化能力差，降低生成模型的质量。为了解决上述问题，本文提出了一种隐式表示网络（IRNet）。IRNet由两部分组成：三维空间解码和特征提取。首先，在解码模块中采用符号距离函数来更好地拟合隐式表示。此外，还设计了一个多级权重损失函数，使得网络生成模型具有更平坦的曲面和更多的细节。然后，在特征提取网络中加入通道注意模块。它有效地减少了背景对图像的干扰，提高了网络的泛化能力。最后，我们的方法生成单个建筑物的网格模型。实验结果表明，与现有方法相比，该方法具有更好的精度。

索引项：形状重建、建筑物、遥感图像、隐式表示网络

# 1 引文

建筑物是应用最广泛的人工地物，在遥感形状重建任务中具有重要意义。受获取遥感数据成本高的限制，传统的方法[1]、[2]需要两幅或多幅输入图像，不适合在一些遥感资源稀缺的地区进行重建任务。因此，仅使用单一遥感影像（SRSI）的物体级建筑物（SROLB）形状重建方法具有更广泛的适用性。然而，如何从包含有限信息的SRSI中重建精确而详细的模型仍然是一个挑战。

在本文中，提出了一种隐式表示神经网络（IRNet），它将符号距离函数（SDF）与多级重量损失和通道注意模块（IRN-MWL-CA）相结合，可以从SRSI执行SROLB。IRNet由两部分组成：三维空间去编码和特征提取。首先，利用SDF建立从建筑物特征到三维空间的更为合适的映射，实现解码网络的隐式表示。通过提出的多级加权损失函数，所构建的模型具有更平坦和更详细的曲面。然后，在特征提取网络中加入通道注意（CA）模块，抑制图像中背景信息带来的干扰。IRN-MWL-CA方法无需背景去除，即可重建精确、详细的建筑物三维网格模型。同时，提高了网络的泛化能力。它可以应用于不同的输入图像。最后，选择网格作为结果模型的表示形式，与体素和点云相比，网格具有完整的三角形表面结构，视觉效果和细节表达最好。实验表明，与最先进的网络相比，IRNet具有更好的性能。这封信的主要贡献总结如下：

  • IRN-MWL-CA用于从单个遥感图像重建目标层建筑物的三维形状。
  • 采用符号距离函数，在解码网络中设计多级加权损耗函数，使重构模型具有更平坦、更细致的曲面。
  • 特征提取网络中增加了通道注意模块，可以抑制图像中背景的干扰，增强泛化能力。

# 2 方法

为了实现建筑物的形状重建，设计了一种端到端的神经网络。通过对单个图像对和标准化三维建筑模型对网络进行训练，建立隐式映射关系。它可以从单个图像中提取屋顶和立面等清晰的特征，并在长度、宽度和高度上以精确的形状和比例重建建筑模型。该映射还能够预测图像中建筑物的不可见部分，并生成完整的模型。IRN-MWL-CA的具体重建过程和网络结构如图1所示。

# 2.1 隐式表征网络

传统的三维重建算法通常使用显式表示来训练网络，网络直接生成体素、点云或网格。它只能重建具有低质量细节的简单形状。对于遥感中的SROLB任务，精度和细节非常重要。隐式表示网络对三维模型具有较好的表示能力。它可以重建复杂的对象。它将网络拟合的三维空间转化为网络拟合的目标函数，更适合神经网络的训练。

符号距离函数（SDF）是一种新的三维空间模型表示形式。它是一个连续函数，用于为三维空间中的每个点输出该点与最近模型曲面之间的距离值。如果点位于模型内部，则距离值将设置为负值。相反，模型外部点的距离值设置为正值。根据阈值 τ，模型形状可由该空间中的等值面决定。重构模型的集成生成过程可以描述为：

# 2.3 特征提取中的注意模块

在建筑物特征提取过程中，输入图像中的背景信息将被平均提取。背景特征是冗余的，并且在整个特征图中相互混合，干扰解码过程，从而产生较大的错误甚至错误的模型。因此，去除图像中的背景是提高重建精度的有效途径。自动化建筑物提取方法，如实例分割算法已经成熟。然而，它不可避免地带来额外的掩码错误。手动删除背景是一项精确但昂贵的工作**。因此，抑制特征图中的冗余特征，突出建筑特征是一种简洁有效的方法。**

在IRNet中，Resnet18[14]用作主干网，在每个基本剩余块之前和之后插入通道注意（CA）模块。CA的网络结构如图2所示。公式（5）表示其具体计算过程。使用不同的合并操作将原始特征映射压缩为两个1×1×n向量。然后向量通过完全卷积层和求和。sigmoid函数用于调整值范围。最终得到的矢量与原始特征图相乘。通过自适应学习，对有利于重建的特征赋予较高的权重。同时，冗余特征也在减少。完整的特征提取网络将输入图像转换成1×256维向量，作为后续解码网络的输入。

# 3 实验

在本节中，为了评估所提出网络的重建和泛化，我们在两个数据集上进行了实验。第一组数据的一部分用于训练IRNet。第一个数据集的其余部分和整个第二个数据集用于测试。通过给出主观和客观评价结果，证明了该网络的有效性和可靠性。具体实施情况如下。

# 3.1 网络的数据集和参数

网络训练需要大量的构建模型和匹配图像。在第一个数据集中，我们从Google Earth获取具有建筑物物体形状和比例的遥感图像，然后手动绘制212个建筑物网格模型。使用105个模型和420张图像对IRNet进行训练。每个模型匹配四幅图像，其中三幅图像由原始图像旋转生成。其余107幅图像和模型用于测试。在第二部分中，我们使用了ISPRS UAV数据集中的213幅建筑物图像[15]，并绘制了匹配的建筑物模型，这些模型也用于测试。

实验在ubuntu 18.04下进行；硬件环境包括两个Intel Xeon Silver 4210CPU和一个使用NVIDIA RTX2080 Ti的GPU，该处理器具有11GB内存。在训练中，批量大小为64，使用Adam优化器，学习率设置为1e-4，历元为1500。训练模型的采样点数为2048，距离在-0.10.1范围内，重建模型的分辨率为128^3。

# 3.2 评价指标

为了评估IRNet的SROLB，我们使用倒角距离（CD）、土方工程距离（EMD）[16]和F评分[17]来评估重建模型。CD和EMD都是距离度量。由于重建模型是归一化的，因此计算结果是无单位的。F-Score用于评估模型细节。它是重建模型上满足重建精细度要求的采样点的比例。在下面的评估中，上述指标是用2048的抽样点数计算的，F分数中的τ=0.08。

# 3.3 结果分析

在实验部分，比较了Pixel2Mesh和ONet，并进行了烧蚀研究。对具有原始损耗的IRNet、具有多级失重的IRNet（IRN-MWL）和集成IRN-MWL-CA进行了测试，以验证每个拟议模块的有效性和重构能力。在第一个实验中，测试了107个单独的建筑图像。五个典型结果如图3所示。对于Pixel2Mesh，由其重建的建筑模型过于圆整。由于背景信息的干扰，很难预测准确的形状，尤其是建筑物（d）（e）中的额外投影。通过ONet重建的模型有了显著的改进。但它仍然生成了错误的建筑模型（c），并且无法恢复建筑（d）（e）中的小型结构。相反，IRN-MWL-CA重建了精确的形状。它表面平整，边缘锋利，符合人工建筑的特点。随着所提出模块的增加，结果模型逐渐完善。对于建筑物（d）和（e），没有MWL和CA模块的原始网络无法在ONet中获得巨大优势，但两个设计模块带来了可观的性能增益。表1给出了测试集中107幅图像的平均客观评估结果。在所有三种不同的评估指标中，所提出的网络获得最佳结果。
