# 基于基本形状及其拓扑关系的点云建筑物重建方法

作者：郝雯 期刊：西安理工大学学报

数据：地面激光扫描仪

# 1 建筑物墙面的提取

# 1.1 点云分割

主成分分析法 → 计算法向量 → Mean shift聚类 → 高斯映射数据集 → 平面点云簇的识别

Mean shift聚类方法与K均值聚类相比：

Mean shift算法不需要事先设定分类的数目ｋ

# 1.2 建筑物墙面提取

平面点云簇的识别 → 建筑物墙面，遵循以下规则：

  1. 地面法向量垂直向上
  2. 墙面法向量与地面法向量乘积约等于0
  3. 墙面有一定高度
  4. 墙面有一定面积

# 2 拓扑关系分析

面面之间的拓扑关系总结为８种：相离、相接、相叠、相等、包含、被含、覆盖和被覆盖。（本文不考虑相叠、相等）

通过设置规则，监测面与面之间的拓扑关系
