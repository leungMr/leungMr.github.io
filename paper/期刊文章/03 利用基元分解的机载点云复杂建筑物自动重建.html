# 利用基元分解的机载点云复杂建筑物自动重建

赵赛男 遥感信息

研究内容

  • 对屋顶面构建拓扑图并依据最小环原则进行拓扑分解，将复杂建筑物分解成多个简单基元
  • 对基元中的面加入约束条件，进行最小二乘法拟合
  • 基元模型边界、顶点的计算，进而生成模型

关联学位论文：

  • 基于机载LiDAR的规则建筑物自动重建研究

# 1 原理和方法

# 1.1 数据预处理

数据预处理包括平面分割和屋顶面轮廓线提取。

  • 建筑物屋顶面的分割算法：区域增长算法、聚类算法、3D霍夫变换、RANSAC算法、等

  • a-shaps算法用来提取各个屋顶面的离散边界点

# 1.2 拓扑图分解与几何约束

# 1.2.1 拓扑图生成与分解

建筑物的拓扑图包含3个基本要素：

  • 独立点：不与其他屋顶面相邻的屋顶面
  • 独立边：代表２个屋顶面之间的相邻关系
  • 最小独立环：代表环中各个屋顶面相交于一个点

# 1.2.2 基元库

基元库的类型（大类）：

  • 点式：即单独存在的建筑
  • 边式：可分为对称平面对和非对称平面对

# 1.2.3 最小二乘法拟合约束

根据基元的拓扑子图以及屋顶面法向量之间的关系引进几何约束，涉及的约束类型及判断条件如下：

  1. 若单个屋顶面的法向量与地面近似垂直，则屋顶面水平；
  2. 若两屋顶面对称，则它们的坡度相同；
  3. 若两屋顶面相邻且相交线的方向向量与地面近似平行，则相交线水平；
  4. 若基元的拓扑子图为环式，多个屋顶面交于一点。

# 1.3 几何重建

# 1.3.1 基元模型生成

目的：将复杂的屋顶面，分解为简单的规则基元，并将基元模型的几何要素归纳为４类：屋顶交点与边界角点、屋脊线与边界线。其中：

  • 屋顶交点和屋脊线： 根据基元间的拓扑关系，利用屋顶面相交求得。
  • 边界线：采用建筑物整体外轮廓结合屋脊线的方法
  • 边界角点：通过边界线拟合的辅助面与屋顶面相交求 得

具体步骤：

  1. 用a-shapes算法提取轮廓点，连接成小线段，并进行分组，精化。该边界线称为一类边界线；
  2. 使用屋脊线充当二类边界线；
  3. 以上2类初始边界线确定后，通过将边界线拟合的辅助面与屋顶面相交可求得边界角点

在求解边界角点的过程中，将其分为3类：

  • 位于屋脊线上的边界角点：Ａ 类点，利用相邻的２条边界线拟合一个辅助面并与屋脊线相交

  • 非屋脊线上与 Ａ 类点相邻的边界角点：B类点，将相邻的2条边界线分别作一个垂面，与该点所在的屋顶面相交

  • 非对称面上的的边界角点：C类点，方法基本同上
