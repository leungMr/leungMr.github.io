# Soft Rasterizer: A Differentiable Renderer for Image-based 3D Reasoning

**摘要：**渲染通过模拟图像形成的物理过程，架起了2D视觉和3D场景之间的桥梁。通过反转这种渲染器，可以想到一种从2D图像推断3D信息的学习方法。然而，标准图形渲染器涉及一个基本的离散化步骤，称为光栅化，这可以防止渲染过程是可微分的，因此可以进行渲染。与最先进的可微渲染器[29,19]不同，它只在反向传播中近似渲染梯度，我们提出了一个真正可微的渲染框架，该框架能够（1）使用可微函数直接渲染彩色网格，（2）从各种形式的图像表示（包括轮廓、着色和彩色图像）向网格顶点及其属性反向传播有效的监控信号。**我们框架的关键是一个新的公式，该公式将渲染视为一个聚合函数，它融合了所有网格三角形相对于渲染像素的概率贡献。**这样的公式使我们的框架能够将梯度流到遮挡顶点和远距离顶点，这是以前的技术状态无法实现的。我们表明，通过使用所提出的渲染器，可以在三维无监督单视图重建中实现定性和定量的显著改进。实验还表明，我们的方法能够处理基于图像的形状拟合中的挑战性任务，这对于现有的可微渲染器来说是不平凡的。代码可在https://github.com/ShichenLiu/SoftRas.

# 1 简介

从二维图像中理解和重建三维场景和结构一直是计算机视觉的基本目标之一。基于图像的三维推理的关键是找到从像素到三维属性的足够监督。为了获得图像到三维的相关性，先前的方法主要依赖于基于二维关键点/轮廓[3,35,26,32]或形状/外观先验[1,28,6,23,48]的匹配损失。然而，上述方法要么局限于任务特定的领域，要么由于二维特征的稀疏性，只能提供较弱的监督。相反，作为从3D资源生成2D图像的过程，渲染通过模拟图像形成的物理机制将每个像素与3D参数联系起来。因此，通过反转渲染器，可以获得通用3D推理任务的密集像素级监督，这是传统方法无法实现的。

然而，在传统的图形管道中，渲染过程是不可区分的。**特别是，标准网格渲染器涉及称为光栅化的离散采样操作，该操作可防止梯度流入网格顶点。**由于正向渲染功能是高度非线性和复杂的，为了实现可微分的渲染，最近的进展[29,19]仅使用手工制作的函数近似向后梯度，而在正向过程中直接使用标准图形渲染器。在基于图像的三维重建任务中取得了令人满意的结果，但正反向传播之间的不一致可能导致不可控的优化行为，并限制了对其他三维推理任务的泛化能力。我们在第5.2节中指出，在基于图像的形状拟合中，这种机制会导致3D参数无法有效优化的问题情况。

**在本文中，**我们不是研究一种更好的渲染梯度形式，而是研究区分前向渲染函数的关键问题。**具体而言，我们提出了一个真正可微分的渲染框架，该框架能够在正向过程中渲染彩色网格（图1）。此外，我们的框架可以考虑各种3D属性，包括网格几何、顶点属性（颜色、正常等）、相机参数和照明，并且能够从像素到网格顶点及其属性流动有效梯度。**作为一个通用模块，我们的渲染器可以插入神经网络或非学习优化框架，而无需参数调整。

我们的方法的关键是将渲染视为“软”概率过程的新公式。与标准光栅化器不同，标准光栅化器仅选择观察方向上最近三角形的颜色（下图1），我们建议所有三角形对每个渲染像素都有概率贡献，可以将其建模为屏幕空间上的概率贴图。虽然传统的渲染管道以一种热的方式合并着色片段，但我们提出了一个可微聚合函数，该函数基于概率贴图和三角形的相对深度融合每个三角形的颜色贴图，以获得最终渲染结果（图1）。新的聚合机制使我们的渲染器能够将渐变流到所有网格三角形，包括被遮挡的三角形。此外，由于其概率公式，我们的框架可以将监控信号从像素传播到远距离三角形。我们称我们的框架为软光栅化器（SoftRas），因为它“软化”离散光栅化以实现可微性。

由于一致的向前和向后传播，SoftRas能够提供高质量的梯度流，用于监督基于图像的3D推理的各种任务。为了评估SoftRas的性能，我们展示了在3D无监督单视图网格重建和基于图像的形状拟合中的应用（图2，第5.1和5.2节）。特别是，由于SoftRas仅基于渲染损失向网格生成器提供强错误信号，因此可以从单个图像实现网格重建，而无需任何3D监控。为了忠实地对网格进行纹理处理，我们进一步提出了一种从输入图像中提取代表性颜色的新方法，并将颜色回归作为一个分类问题。关于基于图像的形状拟合任务，

我们表明我们的方法能够：

  1. 使用考虑所有三角形概率贡献的聚合机制处理遮挡；；
  2. 与其他可区分渲染器相比，提供更平滑的能量景观，通过使用平滑渲染避免局部极小值（图2左）。

实验结果表明，我们的方法在数量和质量上都明显优于现有的方法。

# 2 相关工作

**可微渲染：**为了将观察图像中的变化与3D形状操纵中的变化联系起来，许多现有技术已经利用了渲染的导数[11,10,30]。最近，OpenDR[29]引入了一种近似可微渲染器，该渲染器生成从投影像素到3D参数的导数。NMR[19]建议使用手工制作的函数来近似光栅化的后向梯度，以实现可微分渲染。最近，Li等人[24]引入了一种可微光线跟踪器，以实现二次渲染效果的可微性。3D人脸重建[38,40,39,41,9]、材质推断[27,7]和其他3D重建任务[49,37,33,14,22,34]的最新进展利用了一些其他形式的可微渲染层来获得神经网络中的梯度流。然而，这些渲染层通常是为特殊目的而设计的，因此不能推广到其他应用程序。

本文重点介绍一个通用的可微渲染框架，该框架能够使用可微函数直接渲染给定网格，而不仅仅是近似反向导数。

**基于图像的三维推理：**二维图像被广泛用作推理三维属性的媒介。尤其是基于图像的重建技术受到了广泛的关注。传统方法主要利用基于多视图几何体的立体对应[13,8]，但仅限于多视图提供的覆盖范围。随着大规模3D形状数据集（5）的可用性，基于学习的方法（43, 12, 15）考虑到从数据中学习到的形状，可以考虑单个或很少的图像。为了简化学习问题，最近的工作通过预测中间2.5D表示来重建3D形状，如深度图[25]、图像集合[18]、位移图[16]或法线图[36,44]。姿态估计是理解视觉环境的另一个关键任务。对于三维刚体姿态估计，早期方法试图将其转化为分类问题[42]，而最近的方法[20,46]可以使用深度神经网络直接回归6D姿态。估计非刚性物体（如人脸或身体）的姿势更具挑战性。通过检测2D关键点，在估计2D姿势方面取得了很大进展[31,4,45]。为了获得3D姿势，在最近的方法[3,4,17,2]中加入了形状先验[1,28]，以最小化形状拟合误差。

我们提出的可微渲染器可以提供对3D属性的密集渲染监控，有利于各种基于图像的3D推理任务。

# 3.4 与以前工作的比较

在本节中，我们将我们的方法与最先进的基于光栅化的差分渲染器进行比较：OpenDR[29]和NMR[19]，梯度流如图5所示。我们在附录A中提供了梯度计算的详细分析。

**从像素到三角形的渐变：**由于OpenDR和NMR都在前向过程中使用标准图形渲染器，因此它们无法控制中间渲染过程，因此无法将渐变流到最终渲染图像中遮挡的三角形（图5（a）左侧和中部）。此外，由于它们的梯度仅在图像平面上工作，OpenDR和NMR都无法优化三角形的深度值z。相反，我们的方法完全控制内部变量，并且能够通过聚合函数将梯度流到不可见三角形和所有三角形的z坐标（图5（a）右侧）。

**屏幕空间从像素到顶点的渐变：**由于我们的连续概率公式，在我们的方法中，屏幕空间中像素pj的梯度可以将梯度流到所有远处的顶点（图5（b）右侧）。然而，对于OpenDR，由于局部过滤操作，顶点只能接收近距离内相邻像素的梯度（图5（b）左）。关于NMR，白色区域内的像素相对于三角形顶点没有定义梯度（图5（b）中间）。相比之下，由于我们的方向不变公式，我们的方法没有这样的问题。

# 4.1.1 彩色重建

我们的颜色生成器没有直接回归颜色值，而是将颜色重建公式化为一个分类问题，学习在每个采样点重用输入图像中的像素颜色。设Nc表示M上的采样点数量，H和W分别为输入图像的高度和宽度。然而，朴素的颜色选择方法的计算成本是禁止的，即O（H W Nc）。为了应对这一挑战，我们提出了一种使用调色板对网格着色的新方法，如图7所示。**具体地说，在将输入图像传递给神经网络之后，提取的特征被馈送到（1）采样网络，该采样网络对用于构建调色板的代表性颜色进行采样；和（2）一个选择网络，该网络将调色板中的颜色组合起来，用于对采样点进行纹理处理。**颜色预测是通过将颜色选择与学习的调色板相乘获得的。我们的方法将计算复杂度降低到O（Nd（HW+Nc）），其中Np是调色板的大小。通过适当设置Np，可以显著降低计算成本，同时实现清晰准确的颜色恢复。

# 4.2 基于图像的形状拟合

基于图像的形状拟合在各种任务中具有根本性的影响，例如姿势估计、形状对齐、基于模型的重建等。但是，如果图像和三维参数之间没有直接的相关性，传统方法必须依赖于粗略的对应，例如2D关节[3]或特征点[35]，获取用于优化的监控信号。相比之下，SoftRas可以直接将像素级错误反向传播到三维特性，实现高质量形状拟合的密集图像到三维对应。然而，一个可微渲染器必须解决两个难题，以便于应用。（1） 遮挡感知：3D模型的遮挡部分应该能够接收渐变，以便处理较大的姿势变化。（2） 远距离影响：像素处的损失应该对远距离网格顶点产生影响，这对于在优化过程中处理局部极小值至关重要。虽然先前的可微渲染器[19,29]无法满足这两个标准，但我们的方法同时处理了这些挑战。（1） 我们的聚合函数f使用来自所有三角形的概率贴图，使梯度能够流向所有顶点，包括被遮挡的顶点。（2） 我们基于概率分布的软近似允许梯度传播到远端，同时可以很好地控制感受野的大小（图4）。为此，我们的方法可以通过最小化以下能量目标，忠实地解决基于图像的形状拟合问题：
