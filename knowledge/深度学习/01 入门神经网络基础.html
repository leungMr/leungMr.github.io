# 1 神经网络基础

# 1.1 人工神经元

人工神经元

输入 → 权重分配 → 求和 → 激活函数 → 输出结果

# 1.2 多层感知机

输入层 → 隐藏层 → 输出层

Multi Layer Perceptron：单层神经网络的基础上引入一个或多个隐藏层，使神经网络有多个网络层

在隐藏层加入激活函数，可避免网络退化（多层退化为的单层） 。

# 1.3 激活函数

作用：

  1. 让多层感知机成为真正的多层，否则等于一层

  2. 引入非线性，使网络可以逼近任意非线性函数。

    万能逼近定理：只用一层多层感知机+激活函数，可以逼近任意的非线性函数

常用的激活函数：

  • Sigmoid（S型函数）：值域 0 — 1，符合概率分布的形势，RNN中常用

  • Tanh（双曲正切）：值域 -1 — 1，关于均值0对称

  • ReLu（修正线性单元）：BP、CNN中常用

    **当落入饱和区，**激活函数的导函数趋于0，梯度为0，不利于传播（逆向修正）

# 1.4 反向传播

**前向传播：**输入层的数据从开始从前向后，数据逐步传递至输出层

**反向传播：**损失函数开始从后向前，梯度逐步传递值第一层

**反向传播作用：**权重的更新，使用网络输出更接近标签

**损失函数：**衡量模型输出与真实标签的差异

**反向传播原理：**微积分中的链式求导法则

梯度下降法：权值沿梯度负方向更新，使函数值减小

导数：函数在指定坐标轴上的变化率

方向导数：函数在指定方向上的变化率

梯度：一个向量（矢量），方向为方向导数取得最大值的方向

学习率（Learning Rate，LR）： 控制更新步长 ，（沿着梯度负方向更新）

# 1.5 损失函数

**损失函数：**衡量模型输出与真实的标签之间的差距（单样本）

代价函数：所有样本损失函数的平均值（总样本，数据集）

**目标函数：**代价函数+正则项

两种常见的损失函数

  1. 均方误差（Mean Squared Error，MSE）：输出与标签值之差的平方的均值（常用于回归任务）
  2. 交叉熵（Cross Entropy，CE）：交叉熵源自信息论，用于衡量两个分布的差异（常用于分类任务）

**Softmax函数：**将数据变换到符合概率分布的形式

# 1.6 权值初始化

**权值初始化：**训练前对权值参数赋值，良好的权值初始化有利于模型训练

方法：

  • Xavier初始化
  • Kaiming初始化（也称为MSRA初始化）

# 1.7 正则化方法

**Regularization：**减小方差的策略，通俗理解为减轻过拟合的策略

误差可分解为：偏差，方差，与噪声之和。

  • 偏差：度量了学习算法的期望预测与真实结果的偏离程度，即刻画了学习算法本身的拟合能力
  • 方差：度量了同样大小的训练集的变动所导致的学习性能的变化，即刻画了数据扰动所造成的影响
  • 噪声：表达了在当前任务上任何学习算法所能达到的期望泛化误差的下届

过拟合现象：训练集表现良好，在测试集表现糟糕

正则化方法：

  1. L1、L2
  2. Dropout：随即失活
  3. ...

# 2.1 发展历史

第一个卷积网络雏形 — 新认知机（Neocognitron）（1980年）

第一个大规模商用卷积神经网络 — Lenet5（1998年）

第一个技惊四座的卷积神经网络 — AlexNet（2012年）

AlexNet成功的关键要素：算料（ImageNet）+ 算力（GPU）+ 算法（AlexNet）

# 2.2 卷积层

图像识别特点

  • 特征具有局部性：老虎重要特征”王字“
  • 特征可能出现在图像中的任何位置
  • 下采样图像，不会改变图像目标

**卷积核：**具有学习参数的算子，用于对输入图像进行特征提取，输出通常称为特征图（feature maps）

填充（Padding）：在输入图像的周围添加额外的行/列

步长（Stride）：卷积核滑动的行数和列数称为步长，控制输出特征图的大小，会被缩小为1/s倍

**多通道卷积：**RGB图像时3·h·w的三维的数据，第一个维度3，表示channel，通道数

# 2.3 池化层

Pooling layer

池化：一个像素表示一块区域的像素值，降低图像的分辨率

  • MaxPooling：取最大值
  • AveragePoling：取平均值

池化作用：

  1. 环节卷积层对位置的过度敏感
  2. 减少冗余
  3. 降低图像分辨率，从而减少参数量
