# 倾斜相机的分类

  1. 扫摆式相机系统

    垂直于飞行方向，左右扫摆获取不同视角的影像

  2. 马耳他十字结构相机系统

    最常见的航摄仪结构，相机倾角约45°

  3. 组合宽角相机系统

    相机倾角一般30°左右，同时拍摄的5张照片有一定的重叠度

# 倾斜摄影的特点

  1. 影像多于垂直摄影测量
  2. 从不同角度获取影像，会导致几何变形，传统数据处理系统不再适用
  3. 不仅可以得到顶部纹理信息，还可以得到侧面纹理信息
  4. 倾斜航摄仪一般搭载在低空飞行器，获取影像分辨率更高
  5. 通过影像外方位元素可以测量影像中的物体的高度、角度、坡度等信息
  6. 降低了三维建模的时间成本和劳动成本，促进社会城市的发展，从二维到三维的转化
  7. 能获取地物各个角度的影像，可以提高识别物体的能力，如路灯、电线杆

# 倾斜摄影数据处理流程

数据采集 → 惯导数据采集 → 五角系统处理目录 → 质量控制和影像质量检查 → 影像纠正 → 准确性质量检查和同名点量测 → 处理质量检查 → 正射校正 → 大地面镶嵌 → 授权 → 创建数据库 → 创建标题 → 数据仓库质量检查 → 服务分发（杨久勇，2010）

# 倾斜摄影测量的应用

  1. 数字城市建设
  2. 旅游景点推广
  3. 安全运营管理
  4. 文化遗产的保护和重建
  5. 地震灾害监测

# 航线的设计

  1. 保证低点的分辨率和高点的重叠度
  2. 根据影像分辨率计算相对航高
  3. 根据航向重叠率计算基线
  4. 根据旁向重叠率计算航摄间隔

# 无人机建模关键技术

  1. 特征提取

    因为不同相片（同一区域）在拍摄时候相机有不同的倾斜角度，导致影像具有不同程度的变形，使用灰度相关进行匹配同名点受到一定的局限性。

    SURF算法更适用于透视变换较为严重的倾斜影像匹配。

  2. 密集匹配技术

    1. 借助特征点在多幅影像之间进行匹配，得到稀疏的面片；
    2. 扩展初始匹配点，在临近区域匹配更密集的像素点和面片；
    3. 发现并剔除上一步中出现的匹配错误，去除光度一致性差和几何一致性弱的面片。

    基于以上步骤，可以获取超高密度的点云的实景数字表面模型（DSM）数据

  3. 纹理映射技术

    将二维纹理图像和三维模型进行配准并贴图。
